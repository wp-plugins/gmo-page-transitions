== Description ==

GMO Page Transitions adds Page Transitions actions to your site. 
Click on the link, and page will slide over to left or right. This effect will not apply when "target=_brank" is used.

== Installation ==

Search and download plugin from either WordPress admin page or http://wordpress.org/plugins.  
From the WordPress admin page, simply activate the plugin, or upload a file from "Add New" to install and activate plug-in.  

== Instruction ==

1. Install plugin and activate.  Open [GMO Page Transitions] in the settings. 
2. Select effect from "slide settings". 2 effects are available as listed below.
3. click "Save Changes"
